# Flat CSS
Userstyle/CSS snippet to remove border-radius and flatten all UI.

## 🛠 Installation
1. Install the [Stylus browser extension](https://add0n.com/stylus.html).
2. Click the **install button above** to install the fix directly.

> **Note:**
> Some sites may apply Flat CSS to exceeding elements unintentionally and may look worse or broken.

## 📁 Other ways to install
For userstyle managers other than Stylus or a custom-compiled version, please refer to the source code or use your manager's "Import" function with the raw URL.
